# owltouch

**How to download & install :**
Requirement : 
* [NodeJS](https://nodejs.org/en/)
* [Yarn](https://yarnpkg.com/en/docs/install#windows-stable)
```
git clone https://github.com/Drigtime/owltouch
cd owltouch && yarn
yarn electron .
```
